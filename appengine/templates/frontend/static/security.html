Seguridad

Algunas de las medidas de seguridad que hemos tomado:

  • ALMACENAMIENTO OFFLINE - El 100% de los monederos (y las claves privadas) se almacenan fuera de línea (desconectado de Internet) para evitar el robo o pérdida. Esto significa que ante un hackeo total del sitio, y/o acceso a los códigos fuente y/o a la base de datos del sistema no conducirá a un robo de fondos (bitcoins o pesos).
  • TRANSACCIONES OFFLINE - El 100% de los retiros (transacciones fuera del sitio) se realizan de manera artesanal.
    El signado de los retiros de bitcoins se efectúa en computadoras fuera de línea (desconectadas de Internet).
  • CHECKPOINTS DE BALANCES - Previo a la ejecución de los retiros de bitcoins y pesos verificamos la consistencia del balance rearmando el historial de compras, ventas, depósitos y retiros desde el último checkpoint.
  • OTP AUTH - Ofrecemos la autenticación de dos pasos OTPauth para el acceso al sitio.
  • SSL - El sitio funciona por completo a través de SSL (https).
  • DATOS ENCRIPTADOS - Ciframos las contraseñas almacenadas en la base de datos.
  • NO SQL - No utilizamos SQL. Eliminamos así la vulnerabilidad de SQL Injection.
  • STRONG PASSWORD - Comprobamos la seguridad de las contraseñas al crear una cuenta y al modifcarlas.
  • SEGURIDAD - Estamos bajo auditoría de seguridad + compliance de infraestructura para prevenir vulnerabilidades del sistema.
